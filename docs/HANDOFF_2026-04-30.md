## 5) Continue on another PC

1. Clone and checkout:
```bash
git clone https://github.com/Aiden-lee999/mediAI.git
cd mediAI
git checkout main
```

2. Install deps:
```bash
npm install
```

3. Set environment variables (`.env`, `.env.local`, `.env.production`) for:
- Database/Prisma connection
- OpenAI key
- Any public API keys used by sync scripts

4. Verify app:
```bash
npm run build
npm run dev
```

5. Resume reimbursement work with:
```bash
npx --yes tsx scripts/check_unknown_reimbursement.ts
npx --yes tsx scripts/reimbursement_distribution.ts
```
