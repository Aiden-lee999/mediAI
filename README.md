# Doctor AI Assistant (MedEvidence MVP)

의사가 진료 및 연구 중 겪는 정보 과부하를 해결하기 위한 의사 전용 근거 기반 AI 플랫폼 (MVP).

## 🚀 프로젝트 개요
* **타겟**: 면허 인증을 거친 국내 의사 (개원의, 전공의 등)
* **핵심 가치**: "의사는 무료, 빠르고 검증된 수치/논문 직접 제공"
* **수익 모델**: 
  * 제약사/의학회 Sponsor 기반 학술 콘텐츠 (규제 친화적 분리형 UI 적용)
  * 비식별/집계 기반 의사 검색 트렌드 인사이트 판매
* **기능**: 임상 질의응답 (가이드라인/논문 기반), 약물 상호작용 및 금기 확인 기능

## 💻 실행 방법
본 프로젝트는 순수 HTML, CSS, Vanilla JS 로 구성되어 있어 
추가적인 서버 세팅이나 라이브러리 설치 없이 즉시 실행 가능합니다.

1. `index.html` 파일을 브라우저(Chrome, Edge 등)에서 엽니다.
2. 메인 검색창에 질의(예: "고혈압 + 당뇨 환자 60대 약 조합")를 입력하거나 아래의 **빠른 검색 태그**를 클릭합니다.
3. 1초 대기 후 근거 기반 답변, 요약, 약물 부작용, 논문 레퍼런스 카드가 렌더링 됩니다.

## 📁 파일 구조
* `index.html` : 뷰 페이지 (메인 레이아웃, 검색창, 우측 스폰서드존 구성)
* `style.css` : 컬러 시스템, 반응형 카드 UI 등 전반적 스타일링
* `app.js` : 사용자 입력에 기반하여 미리 작성된 더미 AI 응답을 제공하는 시뮬레이션 코드
